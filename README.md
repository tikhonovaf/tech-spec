# Тестовое задание
При разработке Rest сервисов использован подход API FIRST, то есть сначала создается спецификация OPEN API, 
потом на основе нее генерятся все dto и требуемая обвязка.
На java пишется только уровень service.

Для маппинга использован один из вариантов, принятый у нас в проектах. 
Можно использовать и MapStruct.

Для скачивания докер образов приложения и БД можно использовать
docker-compose.yml  - есть в проекте

Для выполнения сервисов APi на локальной машине можно использовать ссылку
http://localhost:8081/swagger-ui/index.html#/
